# Evaluation

Planned modules:

- `prior_checks.py`: contract and prior predictive checks;
- `realism.py`: scorecards and adversarial real/synthetic metrics;
- `transfer.py`: TSTR, RTSR, few-shot, and benchmark runners;
- `invariance.py`: same-latent/different-view embedding metrics;
- `baselines.py`: PLS, PCA, TabPFN, and existing feature baselines.

Validation failures should stop downstream experiments.
