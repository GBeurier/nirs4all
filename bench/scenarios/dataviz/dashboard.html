Preprocessing, decomposed

The master CSV's preprocessing_pipeline column is heterogeneous — AOM atom-bank labels, classical pp names, HPO compound strings, and stringified Python pipeline paths all coexist. The aggregator now parses each entry into five canonical components (baseline · scaling · scatter · derivative · atom_bank) so we can report on what actually helps, marginalised over everything else.
